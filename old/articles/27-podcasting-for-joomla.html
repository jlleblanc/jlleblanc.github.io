Podcasting for Joomla

A podcasting suite for Joomla can be downloaded here. Be sure to download the component, module, and mambot for full functionality. I had to make changes to the FeedCreator class which are included with the component and will not interfere with your Joomla installation or existing feeds. A separate copy of these modifications can be downloaded here. The release notes can be viewed below...

Joomla Podcasting Component - Beta

Thank you for installing this component! Be sure to install the accompanying module and Mambot. This is a beta copy of the podcasting component and feedback is greatly appreciated. Here are some of the planned features to be seen in future releases:

  • Uploads through FTP assist
  • Automatic publishing of show notes
  • Ability to podcast from certain categories (rather than just frontpage items)

    • What to expect

    When you select Podcast > Manage Clips from the components menu, you will be presented with a list of all the files in the folder being used for podcasts (default is images/stories, this can be changed through the settings). The middle column gives you the tag to use to include a certain file within an article. The published column shows if this tag appears within any article in the system. Eventually, this will be turned into the standard "publish/unpublish" toggle which will allow you to publish a podcast complete with show notes.

    If you want to include a file from an external server, use a tag like this: {enclose http://www.otherserver.com/file23.mp3 8474349 audio/mpeg } . The first parameter is the filename, the second is the file length in bytes, and the third is the encoding type.

    The "Podcast from" and "Specific Category for Podcasting" settings are yet to be implemented as features.
