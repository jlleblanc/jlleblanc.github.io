Podcast Suite, Version 1.1 Stable

Podcast Suite version 1.1 stable is now available for download: component, module, plugin. This is mainly a bug fix release, with the following issues resolved:

  • Added support for <itunes:author>
  • Fixed the iTunes category format
  • Fixed player autostart issues in IE
  • Fixed the issue where RSS feeds were downloading instead of displaying in the browser
  • Enclosures can now appear in both the introtext and fulltext
  • Off-site links now work properly
  • Fixed so that filenames with parenthesis are properly handled
The following features were also added:
  • The module can now display pcast:// links for one-click subscriptions, as well as display an additional link to the actual feed
  • Quicktime is now supported for AAC playback
  • Category pre-selects have been added for iTunes
  • Feeds for separate Joomla categories are now available.
Uninstall the old versions of all three, then reinstall the new copies.
