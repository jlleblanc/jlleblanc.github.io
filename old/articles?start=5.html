Podcast Suite, Version 1.1 Stable

Podcast Suite version 1.1 stable is now available for download: component, module, plugin. This is mainly a bug fix release, with the following issues resolved:

  • Added support for <itunes:author>
  • Fixed the iTunes category format
  • Fixed player autostart issues in IE
  • Fixed the issue where RSS feeds were downloading instead of displaying in the browser
  • Enclosures can now appear in both the introtext and fulltext
  • Off-site links now work properly
  • Fixed so that filenames with parenthesis are properly handled
The following features were also added:
  • The module can now display pcast:// links for one-click subscriptions, as well as display an additional link to the actual feed
  • Quicktime is now supported for AAC playback
  • Category pre-selects have been added for iTunes
  • Feeds for separate Joomla categories are now available.
Uninstall the old versions of all three, then reinstall the new copies.

Podcast Suite, Version 1.0 Stable

The Podcast Suite Version 1.0 is ready for download: component, module, and plugin. Thank you all for your testing, feedback, and encouragement! Here are some of the new features for version 1.0:

  • An integrated MP3 player, thanks to Eric Zhang's player
  • ID3 tag recognition, thanks to Tadeu Oliveira's library
  • Adjustments to the module that let you override the link and image
  • {player ...} can now be used to add another instance of the Flash player or download link without adding another enclosure to the RSS feed.


    • If you are upgrading from Beta 4, simply uninstall the old component, module, and plugin and install the new ones. Otherwise, you should backup the podcast table in your database, do the reinstallation, then renew the database with the backup.
       

    Podcast Beta 4, released

    Beta 4 of the Podcasting Suite is now available for download. Be sure to grab the latest module, component, and bot.

     UPGRADE NOTICE: To upgrade the component without losing your item-specific iTunes tags, open com_podcast_b4.zip, then copy these files to administrator/components/com_podcast.php:

    uninstall.podcast.php
    toolbar.podcast.php
    toolbar.podcast.html.php
    podcast.xml
    install.podcast.php
    info.podcast.php
    admin.podcast.php
    admin.podcast.html.php

     And copy these files to components/com_podcast:

    podcast.php
    podcast.class.php
    feedcreator.class.php

     Uninstall and reinstall the bot and module only.

     The next upgrade will be easier, I promise!

       

    Podcast beta 3, released

    The Podcast suite, beta 3, is now released. Download the mambot, module, and component (only the component has changed, so you can skip downloading the other two if all you want to do is switch out the old component for the new). iTunes support has been added, but due to the nature of Apple's listing policies, this functionality has not been tested. UTF-8 is now the default encoding (which is preferred by Apple).

    Update: Small patch made to the component to eliminate repeating iTunes image tag. (September 19th, 2005)
       

    Updated mp3 player

    An updated copy of the mp3 player has been released. There is now a module as well as a component. You must have the component installed to use the module.

    The mp3 player is designed to pull mp3 files from the root of your Joomla installation. To use subdirectories, simply add them before the filename (example: images/stories/mymp3song.mp3 instead of just mymp3song.mp3).
